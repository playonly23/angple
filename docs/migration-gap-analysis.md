## A. 포인트 시스템

| 기능                         | 현세대                                        | 차세대                       | 상태           |
| ---------------------------- | --------------------------------------------- | ---------------------------- | -------------- |
| 포인트 조회/내역 UI          | `point.php`                                   | `/my/points` 페이지          | ✅             |
| 글쓰기 포인트 적립           | `write_update.php` → `insert_point()`         | Go 핸들러 `v2_points` 테이블 | ⚠️ 이중 테이블 |
| 댓글 포인트 적립             | `write_comment_update.php` → `insert_point()` | Go 핸들러 `v2_points` 테이블 | ⚠️ 이중 테이블 |
| 글 읽기/삭제/다운로드 포인트 | PHP 각 파일                                   | 없음                         | ❌             |
| 포인트 사용/차감             | `insert_use_point()`                          | 없음                         | ❌             |
| 포인트 만료 관리             | `delete_expired_point()`                      | 없음                         | ❌             |
| 포인트 조회 API              | PHP 직접                                      | Go 구현 완료 (g5_point 읽기) | ✅             |

**핵심 문제**: 조회는 Go가 `g5_point`에서 직접 읽으므로 정상 동작. 하지만 **적립/차감은 분리됨** — PHP는 `g5_point`에 쓰고, Go는 `v2_points`에 씀 → 글쓰기/댓글 포인트가 포인트 내역에 안 보임.
